# 📚 LMS Portal  
## Learning Management System – File Sharing Platform  

**Developed Solo by: Jaideep Singh**  
Built using **ASP.NET MVC (C#)** and **SQL Server**

---

# 🚀 Project Overview

The **LMS Portal** is a scalable, database-driven web application designed to allow students and faculty to **upload, store, and share academic materials** efficiently.

The core functionality focuses on:

- 📄 Uploading SLM (Self Learning Material)
- 📑 Uploading PDFs
- 🖼 Uploading Images/Photos
- 🔗 Automatic Public Link Generation
- 👥 Student-to-Student File Sharing

The system centralizes academic resource sharing and enables seamless distribution of study materials through secure public URLs.

---

# 🎯 Problem Statement

Students often face issues such as:

- Difficulty accessing shared academic materials
- Scattered file sharing across multiple platforms
- Lack of centralized content repository
- No structured link-based academic sharing system

There was a need for a simple, secure, and scalable portal where users could upload files and instantly generate a shareable public link.

---

# 💡 Solution

The LMS Portal provides:

- A centralized academic file repository
- Secure file upload system
- Public link generation for each uploaded file
- Easy sharing among students
- Database-driven file tracking and management

Each uploaded file generates a **unique public URL** that can be shared with other students for direct access.

---

# 🏗 System Architecture

The application follows the **ASP.NET MVC architecture pattern**:

- **Model** – Handles database and file metadata
- **View** – Razor-based UI for file upload and management
- **Controller** – Manages upload logic and link generation

### Architecture Highlights

- Entity Framework ORM
- SQL Server relational database
- Layered architecture
- Secure file handling
- Scalable design for high concurrent users
